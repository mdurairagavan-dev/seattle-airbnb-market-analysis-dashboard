# Seattle Airbnb Market Analysis Dashboard

## Overview

This project presents an interactive Tableau dashboard that analyzes Seattle Airbnb listings. The dashboard provides insights into pricing, revenue trends, bedroom distribution, and geographical patterns to better understand the Airbnb market in Seattle.

## Project Objectives

* Analyze Airbnb listing prices across different Seattle zip codes.
* Identify revenue trends over time.
* Compare average prices based on the number of bedrooms.
* Visualize the geographical distribution of Airbnb listings.
* Develop an interactive dashboard for business analysis.

## Dashboard Features

* Average Price by Zip Code
* Revenue Trend Over Time
* Average Price by Number of Bedrooms
* Bedroom Listings Distribution
* Interactive Zip Code Filter

## Key Insights

* Airbnb prices vary across different Seattle zip codes.
* Properties with more bedrooms generally have higher average prices.
* Revenue trends indicate changes in market performance over time.
* Geographic analysis highlights pricing differences across neighborhoods.
